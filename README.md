## 🚀 How to Use

### Option A — Use directly in a browser
Just open any `.html` file in your browser. No build tools needed.

### Option B — Deploy to GitHub Pages
1. Go to your repo → **Settings → Pages**
2. Set source to `main` branch, `/ (root)` folder
3. Your site will be live at `https://yourusername.github.io/summer-sip/`

### Option C — Integrate into an existing website
Copy individual sections from each HTML file into your existing layout. All styles are self-contained within each file.
